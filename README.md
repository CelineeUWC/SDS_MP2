All artist data was collected from the
[**MusicBrainz API**](https://musicbrainz.org/doc/MusicBrainz_API), an open
music encyclopedia that records metadata about artists, releases, and
recordings worldwide. This dataset was constructed by manually selecting 96 artists to query through the MusicBrainz API. As a result, findings reflect patterns within this curated sample and cannot be generalized to the broader global music landscape. Future work could use a random or stratified sample to reduce selection bias.
